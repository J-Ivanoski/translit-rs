# Translit

Library for transliteration for cyrillic languages.

# Supported formats

## Russian language

1. GOST 7.79 System B, modified ISO 9:1995
2. Passport (2013), ICAO

## Belarusian language

1. GOST 7.79 System B, modified ISO 9:1995

## Ukrainian language

1. GOST 7.79 System B, modified ISO 9:1995
